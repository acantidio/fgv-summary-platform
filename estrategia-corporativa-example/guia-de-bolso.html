05 5 Forças de Porter — Respostas-modelo

Para cada força: exemplo + força/fraqueza + ação (verbo no infinitivo). Estes modelos funcionam para qualquer setor.

Concorrentes

Exemplo genérico: Empresas tradicionais do setor + novos players digitais

F: Rivalidade estimula inovação · Fr: Pressão nos preços

Diferenciar a proposta de valor para competir por qualidade, não por preço

Clientes

Exemplo genérico: Clientes exigentes, informados, com poder de troca

F: Alta disposição a pagar por valor · Fr: Alto poder de barganha

Criar programa de fidelização e elevar custo de troca

Fornecedores

Exemplo genérico: Fornecedores de tecnologia / insumos especializados

F: Alta capacidade técnica · Fr: Dependência / risco de ruptura

Diversificar base de fornecedores e negociar contratos de longo prazo

Substitutos

Exemplo genérico: Soluções alternativas de menor custo / modelo diferente

F: Custo menor / acessibilidade · Fr: Menor qualidade / personalização

Reforçar os diferenciais que o substituto não entrega (qualidade, experiência)

Novos entrantes

Exemplo genérico: Empresa com capital para entrar no mercado rapidamente

F: Capital forte / velocidade · Fr: Falta de experiência / relacionamento

Fortalecer barreiras de entrada (marca, patentes, relacionamento, exclusividades)

07 OKR — Objetivos inspiradores + KRs universais
Objetivos inspiradores (adapte ao caso)
O tom é motivacional, não gerencial. Sem números no objetivo.
Objetivos-modelo
Tornar a [empresa] sinônimo de excelência em [setor]
Fazer cada cliente se tornar um embaixador da marca
Construir uma equipe que o mercado inteiro queira contratar
Transformar nossa operação em referência de eficiência
Dominar o mercado de [região] com uma proposta que ninguém copia
Criar uma experiência que os clientes não encontram em nenhum concorrente
KRs universais (DE → PARA)
NPS: de X para Y pontos
Taxa de retenção: de X% para Y%
Nº de publicações/menções: de X para Y
Taxa de conversão: de X% para Y%
Nº de eventos/parcerias: de X para Y
Turnover: de X% para Y%
Horas de treinamento: de X para Y h/pessoa
Receita de novo canal: de R$X para R$Y

Diferença BSC × OKR: BSC diz "Melhorar satisfação dos clientes" (gerencial). OKR diz "Fazer cada cliente se tornar um embaixador da marca" (inspirador). O conteúdo pode ser o mesmo — o tom muda.

★ Regras de ouro — Checklist antes de entregar
Crítico

Todos os objetivos e ações começam com VERBO NO INFINITIVO (-ar, -er, -ir). Se termina em -ção, -mento, -ância → reescreva.

Crítico

SWOT: rastreabilidade obrigatória (S01, W01, O01, T01). Cada cruzamento cita as tags de origem.

Crítico

BSC: Mapa + Painel. Mapa conecta de baixo para cima. Painel tem 4 colunas: Objetivo, Indicador, Alvo (atual + metas intermediárias + meta final), Projeto.

Importante

Alvos do BSC: SEMPRE com valor atual + metas intermediárias por período + meta final. Nunca só atual → final.

Importante

OKR: Objetivo é inspirador e qualitativo (sem números). Key Results são quantitativos com DE → PARA + unidade. São 3 KRs.

Importante

Visão MUDA (3–5 anos). Missão NÃO MUDA. Valores: 3–5, sem sobreposição.

Dica

Proposta de Valor: preencha o lado do Cliente PRIMEIRO (necessidades, desejos, medos), depois o Produto com tags, depois o Encaixe.

Dica

Leia o enunciado DUAS vezes. Se pede um pilar específico, responda naquele pilar. Resposta fora do pedido = nota zero na questão.
