# Estratégia Corporativa e de Negócios

Material de estudo para a disciplina de Estratégia Corporativa e de Negócios.

## 📚 Conteúdo

### [Material de Estudo](material-de-estudo.html)
Teoria completa de todos os frameworks com exemplos práticos:
- Proposta de Valor (Business Model Canvas)
- Visão, Missão e Valores
- Estratégia Diamante
- Análise SWOT (com cruzamentos)
- 5 Forças de Porter
- BSC — Balanced Scorecard (Mapa + Painel)
- OKR — Objectives and Key Results

### [Guia de Bolso](guia-de-bolso.html)
Respostas universais para qualquer estudo de caso na prova:
- Objetivos estratégicos por pilar (todos com verbo no infinitivo)
- Indicadores universais para o BSC
- Banco de forças, fraquezas, oportunidades e ameaças
- Ações-modelo para os 4 cruzamentos SWOT
- Templates de Visão, Missão e Valores
- Objetivos inspiradores e KRs para OKR

## 🚀 Como acessar

O site está publicado via GitHub Pages:

**https://SEU_USUARIO.github.io/estrategia-corporativa/**

## 🛠 Como publicar

1. Crie um repositório no GitHub chamado `estrategia-corporativa`
2. Faça push de todos os arquivos
3. Vá em **Settings → Pages → Source: Deploy from a branch → Branch: main → / (root)**
4. Em ~1 minuto, o site estará disponível
